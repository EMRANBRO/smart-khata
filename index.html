⚙️ Firebase Setup করুন (একবারই)

  1. console.firebase.google.com এ যান → New Project
  2. Project তৈরি হলে Web App (</>) যোগ করুন
  3. Authentication → Sign-in method → Google → Enable
  4. Firestore Database → Create → Test mode
  5. এই HTML ফাইলে firebaseConfig এ আপনার config paste করুন
  6. Config দেওয়া হলে এই box নিজেই হারিয়ে যাবে ✅
